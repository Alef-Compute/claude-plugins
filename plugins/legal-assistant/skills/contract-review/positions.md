# Clause Positions Reference

## Clause Types to Review

| Clause | Red Flags |
|--------|-----------|
| Liability | Unlimited, <12mo fees, asymmetric |
| Indemnification | Unilateral, uncapped, covers own negligence |
| Termination | No for-cause, <30 day notice, auto-renewal traps |
| Payment | On signing, >1.5%/mo late fees |
| Confidentiality | Perpetual, no standard carve-outs |
| IP | Assignment to vendor, work-for-hire without license-back |
| Governing Law | Foreign jurisdiction, mandatory arbitration |

## When No Playbook Exists

Ask user for their position on each clause type above. Do not assume defaults.
